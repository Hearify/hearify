# 🤖 Hearify Frontend App
Hearify App is a platform that helps you to create quiz questions from your content.

[Deployment Production](https://app.hearify.org) \
[Deployment Test](https://test.hearify.org/)

---

## ⚙️ Technologies
- TypeScript
- React
- SCSS
- Vite

---

## 💻 Installation

### Clone the repository
```shell
git clone https://github.com/Hearify/srm-system
```

### Run in development mode
Copy `.env.example` file, fill it out, and rename it to `.env`.

```shell
npm install
npm run prepare
npm run dev
```

### Build for production

```shell
npm run build
```
Builds the app for production to the `dist` folder.

## 🗂 Project Structure (Currently only v2 version)
````
.github/ _____________________________# Github workflows and PR template
.husky/ ______________________________# Husky config files and pre-comit hook
src/ _________________________________# Application source code
  ├─ api/ ________________________________# Classes for API calls
    └─ client.ts _______________________________# Axios client instance and configuration
  ├─ assets/ _____________________________# Static files
    ├─ images/ _________________________________# Images used in the application (usually only .png)
    ├─ icons/ __________________________________# SVG icons used in the application
    └─ translations.json _______________________# Translations file
  ├─ components/ _________________________# All UI reusable components (e.g. Button, Input, etc.). Cannot contain business logic!
    └─ Component/ ______________________________# Example component (replace with your own)
      ├─ Component.tsx _____________________________# Component view
      ├─ Component.scss ____________________________# Component styles (should not be module-scoped, to easily override UI components) 
  ├─ constants/ __________________________# All constants that are used in the application
  ├─ containers/ __________________________# Smart and reusable components that contain business logic
    └─ Container/ _____________________________# Example container (replace with your own)
      ├─ Container.tsx ______________________________# Container view
      ├─ Container.module.scss ______________________# Container styles (only module)
  ├─ styles/ _____________________________# Global SCSS styles and utils
    ├─ themse.scss ____________________________# Theme (color) CSS variables
    └─ helpers/ _______________________________# SCSS helper functions
        ├─ _breakpoints.scss _______________________# Breakpoints mixins for responsive design
        ├─ _mixins.scss ____________________________# Mixins for common styles
        ├─ _typography.scss ________________________# Typography mixins styles
        ├─ _variables.scss _________________________# Global SCSS variables
        └─ _index.scss _____________________________# Forwards all SCSS helper functions
  ├─ templates/ __________________________# Stores individual template, each often representing a specific route
    └─ Template/ _____________________________# Example template (replace with your own)
      ├─ components/ _____________________________# Template-specific React components
      ├─ Template.tsx ______________________________# Template view
      ├─ Template.module.scss ______________________# Template styles (only module)
  ├─ tests/ ______________________________# Stubs data for unit tests
  ├─ types/ ______________________________# Holds all the types and declarations
  ├─ utils/ ______________________________# Consists of some repeatedly used functions that are commonly used in the project
  ├─ env.d.ts ____________________________# Declare environment type in the global namespace
.eslintignore ________________________# Eslint files and folders to ignore
.eslintrc.cjs ________________________# Configuration file for Eslint
.gitignore ___________________________# Git files and folders to ignore
.prettierignore ______________________# Prettier files and folders to ignore
.prettierrc.js _______________________# Configuration file for Prettier
package.json _________________________# Project dependencies and scripts
package-lock.json ____________________# Package lock file autogenerated by npm
tsconfig.json ________________________# Configuration file for TypeScript
tsconfig.node.json ___________________# Configuration file for Node.js
vite.config.ts _______________________# Configuration file for Vite
````
